title: Windows Server 2019 Önizlemesi
---

Microsoft'un da açık kaynak kodlu bilişim dünyasının evrenine yandan yandan yaklaşma çabası mıdır bilinmez "Windows Server 2019" ile konsol arayüzüne dönme çabasında     olmasına hayret ettim. Açık kaynak denilince akla gelen linux mecrası ise yıllardan beri işlerini server tarafında gayet hızlı ve stabil halledebiliyorken linux tarafındaki gibi esnek olmasa da işin 'bash' ını çeken bir powershell imiz komut vermenizi beklemekte...
